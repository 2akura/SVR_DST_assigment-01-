THE DIFFERENCE BETWEEN SERVER AND DESKTOP

We have first to understand why they are two different things. Server satisfy their role in an infrastructure of a system by either running virtually, on bare–metal hardware, or in cloud environment, depending on their setup. They’re optimized to handle specific task, to serve the services that they’re capable of, such as Database server, DHCP server, etc. While desktop is a machine that is dedicated for end–user. The TL;DR for a desktop, is that they’re just any computer. Thus, we can now conclude that server is a node in an infrastructure that have specific role to serve a certain kind of service. As for desktop, they’re just any computer that even your brother’s gaming setup is a desktop.
